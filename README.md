# Datapath-Logistics-End-to-End-Supply-Chain-Analysis
Delivered an end-to-end supply chain analytics project for Datapath Logistics using 180K rows and 50 columns ,Built a Python data pipeline for cleaning , loaded data into SQL Server for EDA, and created  Power BI dashboards with DAX metrics to analyze profitability, customer behavior, and shipping bottlenecks and give business suggestions.
